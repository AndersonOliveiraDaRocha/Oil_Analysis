# Oil_Analysis

Esse projeto é parte do Curso Nanodegree Data Scientist da UDACITY, esse projeto foi aprovado e a graduação adquirida.

Essa análise usou dados públicos disponíveis na GAPMINDER, os dados são históricos sobre a Produção e Consumo de Petróleo em vários países.

Ciente da importância do bloco OPEP (Organização dos Países Produtores Petróleo), a análise contou com várias visões desse grupo.

Existem também destaques sobre movimentos específicos, de cunho geopolítico, como o Segundo Choque do Petróleo.
